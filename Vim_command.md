
## Command of VIM

### Save and Exit

- :q
  - quit the file
- :q!
  - quit the file without save
- :w
  - save the file
- :wq
  - quit and save the file


### Move the Curser

- j
  - go down
- k
  - go up
- l
  - go right
- h 
  - go left
  
### insert mode

- i
  - enter insert mode
- `esc`
  - escape from insert mode
